# Banners

Модуль для вывода изображений в виде слайдера (либо статично) с различными параметрами и поддержкой загрузкой собственных изображений.

Module for displaying images in a slider (or static) with various parameters and support for uploading custom images.

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **1.0.4**
